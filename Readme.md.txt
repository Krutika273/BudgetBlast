# BudgetBuddy 💸

**BudgetBuddy** is a simple web app that helps users compare prices and manage their wishlist, built using Python and Flask.

## 🚀 Features
- Search page for comparing product prices
- Wishlist feature
- Lightweight and fast
- Easy deployment with Flask

## 🛠️ Tech Stack
- Python
- Flask
- HTML/CSS (Jinja templates)

## 🧑‍💻 How to Run Locally
```bash
pip install flask
python main.py
